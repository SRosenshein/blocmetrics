About Blocmetrics

What is Bloc Jams?

  • Bloc Jams is a Spotify-replica music player web app.
  • The code for this project can be found on Github

What is Blocmetrics?

  • Blocmetrics is an Analytics Tracker Dashboard for Bloc Jams.

Blocmetrics currently tracks the following metrics:

  • Landing Page Views
  • Collection Page Views
  • Album Page Views
  • Individual Song Plays
More metrics to come...

Technologies/Libraries Used:

  • AngularJS
  • Firebase
  • Angular-nvd3 (An Angular variation of D3.js and nvd3)
  • Grunt
  • CSS Bootstrap
  • Moment JS
